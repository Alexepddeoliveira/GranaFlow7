# GranaFlow7

Aplicativo Android para controle financeiro pessoal.  
Permite adicionar lançamentos, listar transações, visualizar resumo e abrir links externos.

## Funcionalidades

- Adicionar receitas e despesas
- Listagem de transações com cores e categorias
- Cálculo automático de saldo, entradas e saídas
- Resumo compartilhável
- Tela “Sobre” com link externo
- Banco de dados local usando **Room**

## Estrutura

- `MainActivity` → Tela inicial com atalhos e visão rápida  
- `AddTransactionActivity` → Tela de novo lançamento  
- `ListActivity` + `TransactionFragment` → Lista de transações  
- `SummaryActivity` → Mostra totais e saldo  
- `AboutActivity` → Abertura de link externo

## Tecnologias

- Kotlin  
- Android Studio  
- Room (persistência local)  
- Material Design Components
